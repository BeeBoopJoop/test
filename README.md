# PDF Upload and Preview App

A Flask-based web application that allows users to upload a PDF, preview it, and extract its text.

## Features
- Upload PDF files
- Preview PDFs
- Extract and display text content from PDFs

## Requirements
- Python 3.8+
- Flask
- PyPDF2
